Add event widget to page.
Select category #AutoLighbox.
Click pencil icon to edit lightbox content.
Select date and times to display lightbox.
Start date and end date are required.

Enter text into details box.

To edit existing call-to-action button:
-Select text with mouse.
-Click three dots icon to expand menu.
-Click link icon.
-Edit URL and Text to display fields.
-Click Save.

To create call-to-action button:
-Type button text in details box.
-Select this text with mouse.
-Click three dots icon to expand menu.
-Click link icon.
-Edit URL and Text to display fields.
-Click Save.
-Click B bold icon.

Final step
-Click Style button in left-hand column of page.
-Scroll to event widget and select.
-Click dropdown menu in left-hand column of page.
-Select SPL: Auto Lightbox.
-Save page.
